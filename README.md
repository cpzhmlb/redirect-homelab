# Homelab Index

This is a static dashboard for homelab services, built with Astro.

## Development

- Run `npm run dev` to start the development server.
- Run `npm run build` to build the static site to the `dist/` directory.
- Run `npm run preview` to preview the built site.

## Deployment

The built `dist/` directory can be deployed to any static host, such as Cloudflare Pages, Netlify, or Vercel.

## Structure

- `src/pages/index.astro`: Main dashboard page.
- `src/styles/style.css`: Styles for the dashboard.
- `public/`: Static assets like images, favicons, and web manifest.
